# تقرير مشروع Fantasy-E3dady

هذا المشروع هو منصة متكاملة لإدارة الأنشطة والمهام لمرحلة إعدادي، تهدف إلى تنظيم وتتبع الأداء بشكل تفاعلي ومحفز.

## التقنيات المستخدمة
- **الواجهة الأمامية (Frontend):** تم بناء المشروع باستخدام **React** و **Vite** لضمان سرعة الأداء وسلاسة تجربة المستخدم.
- **التصميم (Styling):** استخدام **Tailwind CSS** لتوفير واجهة مستخدم حديثة ومتجاوبة بشكل كامل.
- **قاعدة البيانات والخدمات السحابية:** الاعتماد على **Firebase** (Firestore, Authentication, Hosting) لإدارة البيانات والمصادقة والرفع.
- **إدارة الحالة:** استخدام **Hooks** و **Context API** لإدارة حالة المستخدم والبيانات بشكل فعال.

## المميزات الرئيسية
1. **نظام المتصدرين (Leaderboard):** عرض ترتيب الأعضاء والفرق بناءً على النقاط المكتسبة، مع إظهار النسب المئوية للأداء.
2. **إدارة المهام (Task Management):** إمكانية إنشاء وتخصيص مهام فردية وجماعية، وتتبع تقدم الإنجاز.
3. **نظام الصلاحيات:** هيكل متدرج للصلاحيات يشمل (سوبر أدمن، أدمن، قائد، عضو) لضمان تنظيم العمل.
4. **تزامن البيانات:** إمكانية العمل في وضع عدم الاتصال (Offline) مع مزامنة البيانات تلقائياً عند توفر الإنترنت.
5. **لوحة تحكم للمسؤولين:** أدوات متطورة لإدارة المستخدمين، الفرق، والأنشطة بكل سهولة.
